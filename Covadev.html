About CovaDev

Bridge the gap between design and implementation

CovaDev is a web-based AI platform that validates whether source code semantically aligns with BPMN process models using LLM-based summarization, fine-tuned sentence embeddings, and cosine similarity matching — automatically identifying matched, missing, and extra functionality.

The platform supports a complete project lifecycle — from BPMN modeling and GitHub integration, to task assignment, developer submission review, AI-agent code generation, and objective developer evaluation across correctness, quality, timeliness, and communication.

How It Works

Project Workflow

An end-to-end automated pipeline from business process model to actionable validation results.

01
Upload BPMN Model
Upload your .bpmn or XML file. CovaDev parses tasks, events, gateways, and sequence flows automatically.
02
Connect GitHub Repository
Link your project repo for automated code retrieval, branch tracking, and PR management.
03
Submit Source Code
Code functions are extracted and indexed automatically from all Python files in the repository.
04
LLM Summarization & Embeddings
Fine-tuned LLM generates intent summaries; Sentence Transformer creates semantic vector embeddings for each summary.
05
Cosine Similarity Matching
Each BPMN task is matched to its closest code function using cosine similarity with a configurable threshold.
06
Classification: Matched · Missing · Extra
Results classified automatically; similarity scores provided for every decision made.
07
Metrics & Reports
Precision, Recall, F1-Score, and Alignment % computed. Export full structured validation reports.
08
Developer Evaluation
Each developer's submissions scored across correctness, quality, timeliness, and communication — objectively.
